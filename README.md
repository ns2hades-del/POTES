# POTES

POTES est une mini application web de messagerie conçue pour chatter entre amis.

## Fonctionnalités
- Création de compte
- Connexion utilisateur
- Liste d'utilisateurs
- Chat simple
- Profil utilisateur

## Technologies utilisées
- HTML
- CSS
- JavaScript
- Node.js
- Express
